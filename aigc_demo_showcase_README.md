## 🚀 如何使用

### 方式 1：在线预览（推荐）

1. 访问 [CodeSandbox](https://codesandbox.io/)
2. 创建新的 React 项目
3. 复制 `aigc_demo_showcase.jsx` 的内容
4. 粘贴到 `App.js`
5. 安装依赖：`lucide-react`
6. 预览效果

### 方式 2：本地运行

```bash
# 创建新的 React 项目
npx create-react-app aigc-showcase
cd aigc-showcase

# 安装依赖
npm install lucide-react

# 替换 src/App.js 为 aigc_demo_showcase.jsx 的内容

# 启动
npm start
```
