# Screenshots

请在这个目录中添加项目截图，用于 README 展示。

建议的截图：

1. `homepage.png` - 首页界面
2. `generation-progress.png` - 生成进度界面
3. `result.png` - 结果展示界面
4. `database.png` - Supabase 数据库截图（可选）

## 如何添加截图

1. 运行项目：`npm run dev`
2. 访问 http://localhost:3000
3. 截图保存到此目录
4. 在 README.md 中引用：`![Screenshot](screenshots/homepage.png)`
